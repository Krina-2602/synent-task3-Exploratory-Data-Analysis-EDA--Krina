# synent-task3-Exploratory-Data-Analysis-EDA--Krina

[Problem Statement]:
The objective of this project is to perform Exploratory Data Analysis (EDA) on a Netflix movies dataset to uncover trends, patterns, and relationships among various movie attributes such as popularity, ratings, budget, revenue, release year, and country of production. The analysis aims to generate meaningful insights that can support data-driven decision-making.

[Dataset Details]:
The dataset contains detailed information about Netflix movies up to 2025, including:
  Title and Content Type
  Release Year
  Popularity Score
  Ratings and Vote Statistics
  Budget and Revenue
  Country of Production
The dataset includes both numerical and categorical features, making it suitable for statistical analysis and visualization.

[Approach]:
The analysis was conducted using Python, Pandas, Matplotlib, and Seaborn.

Key steps included:
 Data exploration to understand dataset structure, data types, and missing values.
 Statistical summary of numerical and categorical variables.
 Correlation analysis using a heatmap to identify relationships between numerical features.
 Trend analysis of movie releases over time.
 Visualization of content type distribution.
 Identification of the top 10 most popular titles.
 Analysis of rating distributions.
 Investigation of the relationship between budget and revenue.
 Examination of popularity versus audience ratings.
 Analysis of the top content-producing countries.

[Results]:
The EDA revealed several important insights:

Netflix content production has increased significantly over the years.
Budget and revenue show a positive relationship, indicating that higher-budget productions often generate greater earnings.
Most movies receive average-to-good ratings, suggesting generally positive audience reception.
Popularity is influenced by audience engagement metrics such as ratings and votes.
A small number of countries contribute a large share of Netflix content production.
Correlation analysis highlighted meaningful relationships among popularity, votes, budget, and revenue.

Overall, the project demonstrates how exploratory data analysis can be used to understand content trends and audience preferences within a large movie dataset.
